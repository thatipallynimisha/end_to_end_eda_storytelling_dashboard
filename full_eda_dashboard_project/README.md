
# End-to-End EDA & Storytelling Dashboard

## Features
- Data Quality Analysis (info, describe, missing values)
- Data Preview, Key statistics (KPIs)
- Univariate Analysis (histograms, skewness, kurtosis)
- Correlation Analysis (heatmap)
- Time Trend Visualization
- Statistical Testing (Shapiro-Wilk Test)
- Interactive Streamlit Dashboard with Filters & Pages
- Export Filtered Data as CSV

## Run
pip install -r requirements.txt
streamlit run dashboard/app.py
